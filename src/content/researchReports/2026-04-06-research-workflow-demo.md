This is a draft example for the new research publishing workflow.

## Core takeaway

Albert should mainly provide:

- a strong title
- a crisp summary
- the actual report body in markdown
- optional inline HTML blocks when he wants richer visual structure

The site should handle the page shell, archive cards, byline treatment, typography, spacing, and default content styling.
